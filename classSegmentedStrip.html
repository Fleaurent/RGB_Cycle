)

NeoPixelHelper constructor when length, pin, pixel type and segments are known at compile-time. -> LED_COUNT, LED_PIN, NEO_GRB + NEO_KHZ800, segment_starts, n_segments.

Parameters
nNumber of RGB-LEDs in strip.
pArduino pin number which will drive the NeoPixel data in.
tPixel type e.g. NEO_GRB+NEO_KHZ800
segment_starts[]array of led numbers: each entry defines a new segment
n_segmentsnumber of segments i.e. length of segments_starts array
Returns
Adafruit_NeoPixel object. Call the begin() function before use.
